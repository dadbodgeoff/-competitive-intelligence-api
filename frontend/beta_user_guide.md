## Using the Analysis Tool

### Starting a New Analysis

1. **From Dashboard**: Click "Analyze Competitors"
2. **Fill Required Information**:
   - **Restaurant Name**: Your restaurant's name (e.g., "Mario's Pizza")
   - **Location**: Full address or "City, State" (e.g., "New York, NY")
   - **Category**: Select from dropdown (Pizza, Burger, Fine Dining, etc.)
   - **Analysis Tier**: 
     - **Free**: Analyzes 2 closest competitors
     - **Premium**: Analyzes 5 competitors (requires upgrade)

3. **Submit**: Click "Analyze Competitors" to start

### Tracking Progress

- **Real-time Updates**: Progress bar shows completion percentage
- **Current Step**: See what the system is currently doing
- **Time Estimate**: Approximate time remaining
- **Typical Duration**: 2-5 minutes depending on location and competitors

### Viewing Results

Results are organized in three tabs:

#### Competitors Tab
- **Competitor List**: Names, ratings, review counts, distances
- **Mobile View**: Card-based layout for easy scrolling
- **Desktop View**: Table format with sortable columns

#### Insights Tab
- **Threat Insights**: Areas where competitors outperform you
- **Opportunity Insights**: Gaps you can exploit
- **Watch Items**: Trends to monitor
- **Confidence Levels**: High, Medium, Low reliability indicators

#### Evidence Tab
- **Review Quotes**: Actual customer reviews supporting each insight
- **Search Function**: Find specific topics or competitors
- **Filter Options**: View by insight type (threats, opportunities, etc.)

### Exporting Data

- **CSV Export**: Download competitor data and insights
- **File Format**: Includes all competitor info and insights with evidence
- **File Name**: Automatically named with your restaurant and date

## Troubleshooting

### Common Issues

**Analysis Won't Start**
- Check that all required fields are filled
- Try a different location format (full address vs city/state)
- Refresh the page and try again

**Progress Stuck**
- Wait 2-3 minutes (some steps take longer)
- Check your internet connection
- Refresh the page to resume tracking

**Results Not Loading**
- Check if analysis completed successfully
- Try refreshing the results page
- Contact support if issue persists

**Mobile Issues**
- Ensure you're using a supported browser (Safari on iOS, Chrome on Android)
- Check that JavaScript is enabled
- Try landscape orientation for better tablet experience
